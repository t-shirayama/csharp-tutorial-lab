## 到達目標

- `Program.cs` の基本構造を説明できる。
- 変数、条件分岐、繰り返し、メソッドを使った小さな処理を書ける。
- コンパイルエラーの場所と意味を読み取れる。
- 入力値の検証や例外の基本的な扱いを説明できる。
- 入出力、スコープ、デバッグを使って小さな不具合を自力で切り分けられる。
- ステートメント、式、プリプロセッサ、tuple、dynamic の入口を説明できる。
- 文字列操作、型変換、日付と時刻、`using` による resource 解放を説明できる。
- 入力、検証、計算、出力を組み合わせた小さなコンソールアプリを書ける。
