# 推奨書籍

## 目的

体系的に学ぶための書籍候補と選び方を整理します。

## 選び方

- C# のバージョンが新しい。
- .NET CLI、テスト、ASP.NET Core など実務要素がある。
- サンプルコードが公開され、実行できる。
- 言語仕様だけでなく設計や保守性に触れている。

## 分野

- C# 入門
- .NET / ASP.NET Core
- テスト
- オブジェクト指向設計
- データベース
- アーキテクチャ

## 運用ルール

読んだ本は、良かった点、難しかった点、どの記事と関連するかを追記します。古い本は、現在の .NET との差分を明記します。
