# Microsoft Learn 学習ルート

## 目的

C# / .NET を学ぶときに優先して参照する Microsoft Learn の入口を整理します。

## 優先リンク

- [.NET documentation](https://learn.microsoft.com/dotnet/)
- [C# documentation](https://learn.microsoft.com/dotnet/csharp/)
- [ASP.NET Core documentation](https://learn.microsoft.com/aspnet/core/)
- [EF Core documentation](https://learn.microsoft.com/ef/core/)
- [.NET architecture guides](https://learn.microsoft.com/dotnet/architecture/)

## 使い方

最初はこのナレッジベースで全体像をつかみ、詳細仕様や最新情報は Microsoft Learn で確認します。実務判断では、外部ブログより一次情報を優先します。
