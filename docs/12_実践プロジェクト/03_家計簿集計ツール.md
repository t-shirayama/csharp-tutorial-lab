# 家計簿集計ツール

## 目的

日付、金額、カテゴリを持つデータを読み込み、月別・カテゴリ別に集計します。

## 作るもの

家計簿データを読み込み、支出合計とカテゴリ別集計を表示するコンソールアプリです。

## 学習ポイント

- `DateTime` / `DateTimeOffset`
- `decimal` による金額計算
- `GroupBy` と `Sum`
- 入力 validation

## 入力例

```text
2026-05-01,Food,1200
2026-05-02,Book,2500
2026-05-03,Food,800
```

## 完了条件

- 月別の合計金額を表示できる。
- カテゴリ別の合計金額を表示できる。
- 不正な日付や金額を検出できる。

## 発展課題

- CSV ファイルから読み込む。
- 集計結果を JSON で出力する。
- テストデータを使って集計ロジックをテストする。
