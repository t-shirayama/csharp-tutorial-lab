## 到達目標

- CLI でビルド、テスト、パッケージ操作ができる。
- 設定値と環境差分を整理できる。
- CI/CD のログを読み、失敗原因を調査できる。
- SDK、ローカルツール、secret、リリース前確認をチームで揃えられる。
- build、test、image、起動、health check の失敗地点を切り分けられる。
