# DI 登録漏れ

## 症状

`Unable to resolve service for type ... while attempting to activate ...` のようなエラーが出ます。

## 主な原因

- interface と実装を DI に登録していない。
- lifetime が不適切。
- constructor に登録されていない型を追加した。
- project 分割後に登録 extension method を呼んでいない。

## 確認コマンド

```powershell
dotnet build
dotnet run
```

## 解決手順

1. エラーに出ている `type` と `activate` 対象を読む。
2. `Program.cs` または `AddXxxFeature` に登録があるか確認する。
3. Singleton から Scoped に依存していないか確認する。
4. 起動時に API を叩いて再現確認する。
