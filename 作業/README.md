# 作業フォルダ

## 目的

ナレッジベース本文とは分けて、レビュー結果、対応済みレビュー、ファクトチェック記録を管理します。

## フォルダ

- [レビュー](レビュー/README.md): レビュー中、対応待ち、確認待ちの記録を置く。
- [アーカイブ](アーカイブ/README.md): 対応済み、または完了したレビュー記録を移す。
- [ファクトチェック](ファクトチェック/README.md): 各ファイルごとのファクトチェック日時、根拠、結果を記録する。

## 運用ルール

1. レビューした結果は `作業/レビュー/` に Markdown で保存する。
2. レビュー指摘へ対応したら、該当レビュー記録を `作業/アーカイブ/` に移す。
3. ファクトチェックは対象ファイルごとに記録し、確認日時、確認者、根拠リンク、結果を残す。
4. 本文記事を修正した場合は、必要に応じて `INDEX.md`、`ROADMAP.md`、`GLOSSARY.md`、`CHANGELOG.md`、カテゴリ `README.md` も更新する。

## ファイル名の付け方

- レビュー: `YYYY-MM-DD_対象_レビュー.md`
- アーカイブ: `YYYY-MM-DD_対象_レビュー_対応済み.md`
- ファクトチェック: `YYYY-MM-DD_対象_ファクトチェック.md`
